## Key tokens used

| Token | Purpose |
|-------|---------|
| `--mui-color-primary` | Button primary variant |
| `--mui-color-danger` | Destructive variant, danger badges |
| `--mui-color-success` | Success badges |
| `--mui-color-surface` | Card-like surfaces, divider default |
| `--mui-radius-md` | Default border radius |
| `--mui-touch-target` | 44px minimum for all interactive elements |
