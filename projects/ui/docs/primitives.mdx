# Primitives

Core building-block components. Import from `@mushilu-san/ui/primitives`.

```ts
import { Button } from '@mushilu-san/ui/primitives';
```

## Components

| Component | Selector | Status |
|-----------|----------|--------|
| Button | `button[muiButton]`, `a[muiButton]` | ✅ Ready |
| Icon | `mui-icon` | 🚧 Planned |
| Badge | `mui-badge` | 🚧 Planned |
| Spinner | `mui-spinner` | 🚧 Planned |
| Divider | `mui-divider` | 🚧 Planned |
| Avatar | `mui-avatar` | 🚧 Planned |

## Design tokens used

- `--mui-color-primary` / `--mui-color-primary-text`
- `--mui-color-danger`
- `--mui-color-surface` / `--mui-color-border`
- `--mui-space-*`
- `--mui-radius-md`
- `--mui-font-weight-medium`
- `--mui-touch-target` (44px minimum on all interactive elements)
