## CVA pattern

All form controls implement `ControlValueAccessor` for Angular forms compatibility:

```html
<mui-checkbox [(ngModel)]="agreed">I agree</mui-checkbox>

<mui-slider [formControl]="volume" [min]="0" [max]="100" />
```
