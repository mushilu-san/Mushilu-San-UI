| Token | Value | Use |
|-------|-------|-----|

| `--mui-z-raised` | 1 | Cards, raised surfaces |
| `--mui-z-sticky` | 100 | Sticky headers |
| `--mui-z-drawer` | 200 | Side drawers |
| `--mui-z-modal` | 300 | Dialogs, modals |
