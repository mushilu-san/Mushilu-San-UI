## Mobile design guidelines

- All touch targets meet the 44px minimum (WCAG 2.5.5).
- Gesture-based components include keyboard alternatives.
- All animations honor `prefers-reduced-motion`.
- Test with the **Mobile (375px)** viewport preset in the toolbar.
