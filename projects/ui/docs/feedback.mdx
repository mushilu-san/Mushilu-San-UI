## Key behaviors

- **Alert** — `role="alert"` for danger/warning, `role="status"` for info/success. Dismissible via close button or Escape.
- **Toast** — Push via `ToastService`. Polite/assertive live regions. Timers pause on hover/focus.
- **Dialog** — Native `<dialog>` with focus trap and return. Closes on Escape and backdrop click (configurable).
- All animated components honor `prefers-reduced-motion`.
