# Mushilu-San UI

A mobile-first Angular component library built with **design tokens**, **zero runtime dependencies**, and **full accessibility** out of the box.

---

## Getting started

```bash
npm install @mushilu-san/ui
```

```typescript
import { provideMushiluUi } from '@mushilu-san/ui';

bootstrapApplication(AppComponent, {
  providers: [provideMushiluUi()],
});
```

Add the token stylesheet to your `styles` array or import it directly:

```css
@import '@mushilu-san/ui/styles/tokens.css';
```

## Principles

| Principle | What it means |
|-----------|--------------|
| **Mobile-first** | Every component meets 44px touch targets and works at 375px width |
| **Token-driven** | All styling uses `--mui-*` semantic tokens — no hardcoded colors |
| **Accessible** | WCAG AA contrast, keyboard navigation, ARIA roles, reduced motion |
| **Zoneless** | Built for Angular's zoneless future — signal-based reactivity only |
| **Tree-shakable** | Secondary entry points per group — import only what you use |

---

## Theme support

Toggle between light and dark themes using the **paintbrush icon** in the toolbar above.

Components automatically adapt via CSS custom properties — no JavaScript theme switching required. Apply themes with the `data-theme` attribute:

```html
<html data-theme="dark">
```

Or let it follow the OS preference automatically.
